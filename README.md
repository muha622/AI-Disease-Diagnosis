A bilingual (Arabic / English) rule-based disease diagnosis web application built as an academic project for the **Cyber Security Fundamentals** course at Al-Nahrain University.

</div>

---

## ✨ Features

- 🔍 **Symptom-based diagnosis** — select symptoms and get instant results
- 🌐 **Bilingual support** — full Arabic (RTL) & English interface
- 📊 **Confidence score** — percentage match with animated progress bar
- ⚠️ **Severity indicator** — Low / Medium / High severity classification
- 💊 **Precautions** — actionable health recommendations per disease
- 🗂️ **Category filters** — organize symptoms by body system
- 🔎 **Real-time search** — filter symptoms instantly as you type
- 📱 **Responsive design** — works on desktop and mobile

## 📋 How to Use

1. **Clone the repo:**
   ```bash
   git clone https://github.com/<your-username>/disease-diagnosis-system.git
   ```
2. Open `disease_diagnosis_v3.html` in your browser
3. Select your symptoms from the list (use search or category tabs)
4. Click **Diagnose** and view your result
5. Toggle 🌐 Arabic / English using the language button
